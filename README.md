# ordercost
--- 

####在公司帮同事点餐，记账是个麻烦事情。这里实现了一个小巧的Android版记账APP。  
---  

V1.01a  
更新如下功能  
1. 交易记录左滑或者右滑删除  
2. 入账时添加了时间选择按钮

---  

V1.00a
支持如下功能  
1. AA记账，支持返现  
2. 分别记账，支持返现    
3. 入账，入账记录左滑或者右滑删除  
4. 账务一览
